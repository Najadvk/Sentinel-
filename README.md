# 🛡️ Microsoft Sentinel Lab

> Hands-on lab walkthrough covering end-to-end configuration of Microsoft Sentinel as a cloud-native SIEM platform. Built as part of cloud security engineering practice.

---

## 📋 What This Covers

| # | Topic |
|---|---|
| 01 | Setting up a Log Analytics workspace and deploying Sentinel |
| 02 | Connecting data sources — Windows AMA, Azure Activity, Defender for Cloud |
| 03 | Building analytics rules and near-real-time (NRT) detection rules using KQL |
| 04 | Configuring automation rules for automatic incident assignment |
| 05 | Integrating Sentinel with Microsoft Defender XDR for a unified SOC experience |
